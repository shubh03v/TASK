library("pheatmap")
read_counts <- read.table("reads_counts_matrix.txt",row.names = 1,header = TRUE)

Cleaning data

A row should have atleast one read.

keep <- rowSums(read_counts) > 0
read_counts <- read_counts[keep,]

Import metadata

meta <- read.table("metadata.txt", header = TRUE, row.names = 1)

Making heatmap
